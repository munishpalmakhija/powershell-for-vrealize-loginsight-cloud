Add-UserTovRLICloudService

Add's User to vRLI Cloud Service in a particular Org

This cmdlet Add's User to vRLI Cloud Service in a particular Org

Syntax 

Add-UserTovRLICloudService [[-Connection] <PSObject>] [-UserEmail] <String> [[-ServiceRoleName] <String>] [<CommonParameters>]

Parameters

Name Alias Description Required? Pipeline Input Default Value
-Connection vRLIC Connection object false false $defaultvRLICConnection
-UserEmail User Email true false
-ServiceRoleName vRLI Cloud Service Role Name false false log-intelligence:user
